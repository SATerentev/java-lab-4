## Стек технологий

### Backend
- Java
- JDBC
- SQLite
- Maven

### Frontend
- HTML
- CSS
- JavaScript

---

## Запуск проекта

### 1. Создание базы данных

```bash
python script.py
```

### 2. Запуск Backend
Запустить backend-приложение в IDE или Maven

### 3. Запуск Frontend
Открыть файл
```bash
front/index.html
```
в браузере.
