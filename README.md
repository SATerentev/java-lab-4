# Кто хочет стать миллионером

Учебный проект на Java с backend и frontend частью.

## Стек технологий

### Backend
- Java
- JDBC
- SQLite
- Maven

### Frontend
- HTML
- CSS
- JavaScript

---

## Запуск проекта

### 1. Создание базы данных

```bash
python script.py
```

### 2. Запуск Backend
Запустить backend-приложение в IDE или Maven

### 3. Запуск Frontend
Открыть файл
```azure
front/index.html
```
в браузере.

---

## Структура проекта
```azure
project/
│
├── front/          # Frontend часть
```
